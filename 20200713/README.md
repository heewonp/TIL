# Python

## 1. 숫자

- 정수형

  ```python
  a = 1 # a에 1을 대입한다는 뜻.
  print(a)
  ```

- 실수형

  ```
  a = 1.1
  ```

  소숫점이 포함된 숫자를 말한다.

## 2. 연산자(숫자)

- 사칙연산(`+`,`-`,`*`,`/`)

  > 정수와 정수, 정수와 실수 둘다 가능하다.

  ```python
  a = 5
  b = 1.2
  print(a + b) # 6.2
  print(a * b) # 6.0
  print(a - b) # 3.8
  print(a / b) # 4.166666666666667
  ```
